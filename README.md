## Frontend y Backend

```Frontend: VueJS y Bootstrap```
```Backend: NodeJS, MongoDB y Express```


### Requerimientos

```
Tener instalado NodeJS
```

### En el cliente ejecutar lo siguiente

```
npm run serve
```

### En el server ejecutar lo siguiente

```
node index.js
```
